# rakuten_api_book

<h2>◆概要◆</h2><br>
Rakuten Developersの楽天ブック検索APIを活用し、<br>
商品ページまで遷移ができるサイトを構築しました。<br><br>

キーワードを入力し、検索をしたら楽天ブック検索APIに<br>
登録をしている商品を抽出することができます。
